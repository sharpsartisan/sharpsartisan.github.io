This project is an attempt to clone the Weather app for Windows. Some data couldn't be replicated through limitations of the OpenWeather API, but I was able to create a fairly accurate repesentation of the main page.

Some of the skills this project taught me were:

  • How to work with an API
  • Fetch requests and working with async / await
  • Creating a responsive site for both desktop and mobile devices
  • Working with external javascript libraries (Chart.js)
  • Manipulating date and time strings
  • Using CSS Flexbox
  • Writing modular, readable javascript
  • Organising HTML and div grouping
  • Tracking changes and branches using Git

Less 'official' skills I learned include:

  • Googling dozens of stackoverflow answers
  • Writing a bunch of terrible code and immediately deleting it
  • Continuing in the face of insurmountable odds
  • Back stretches

Despite being a single page site, I was really pushed outside my comfort zone when it came to working with the API and keeping track of my code as the html, css, and js files all started to grow.

I feel much more confident in all the areas in the list and I believe this project really helped build my skills and understanding of front-end development.
