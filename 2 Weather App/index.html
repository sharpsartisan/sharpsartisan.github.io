This project is an attempt to clone the Weather app for Windows. Some data (such as moonrise and phase) were not available through the OpenWeather API, but I was able to create a fairly accurate repesentation of the main page otherwise.

Some of the skills this project taught me were:

  • How to work with an API
  • Fetch requests and working with async / await
  • Creating a responsive site for both desktop and mobile devices
  • Working with external javascript libraries (Chart.js)
  • Using the API data to create meaningful graphs
  • Manipulating date and time strings
  • Creating a responsive dropdown list of search results
  • Using CSS Flexbox
  • Writing modular, readable javascript
  • Organising HTML and div grouping
  • Tracking changes and branches using Git

Keeping my code modular and well-commented made it easy to manage even while working on difficult problems.

I feel much more confident in my ability to build a responsive website and I believe this project really helped build my skills and understanding of front-end development.
